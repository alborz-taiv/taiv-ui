# DateTimePicker

[View Mantine Docs](https://v6.mantine.dev/dates/date-time-picker/)

<AIGeneratedBanner />

The DateTimePicker component is a customizable wrapper around Mantine's DateTimePicker component that lets users pick a date and a time in a single input. It shares the same styling conventions as DatePicker and extends Mantine v6's DateTimePickerProps with preset sizing and width options.

### 📦 Import

```tsx
import { DateTimePicker } from '@taiv/ui';
```

### 💡 Best Practices

1. **Provide clear labels**: Describe what date and time the user is picking.
2. **Choose an appropriate format**: Use `valueFormat` to match how users expect dates/times to be displayed.
3. **Handle validation**: Use the `error` prop for invalid or missing values.
4. **Use seconds sparingly**: Only enable `withSeconds` when the extra precision is truly needed.
5. **Consider time zones**: The picker returns local `Date` values — make sure the consuming code normalizes to the timezone your system expects.
