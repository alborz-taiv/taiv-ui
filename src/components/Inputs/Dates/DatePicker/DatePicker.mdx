# DatePicker

[View Mantine Docs](https://v6.mantine.dev/dates/date-picker-input/)

The DatePicker component is a customizable wrapper around Mantine's DatePickerInput component that provides consistent styling and behavior across your application. It extends Mantine v6's DatePickerInputProps while adding custom sizing options and preset styling with a clean calendar interface.

### üì¶ Import


### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what date they should select.

2. **Use appropriate placeholders**: Set meaningful placeholder text that indicates the expected date format or purpose.

3. **Handle validation**: Use the error prop to display validation messages when dates are invalid or required.

4. **Consider accessibility**: Ensure labels are descriptive and consider using description for additional context.

5. **Use consistent sizing**: Choose appropriate sizes based on your form layout and user context.

6. **Icon usage**: Use the calendar icon for better visual recognition, but hide it in compact interfaces if needed.

7. **Date formatting**: Consider the expected date format and provide clear examples or formatting hints.
