# RadioList

The RadioList component is a composite component that renders a list of radio buttons in a styled container. It provides a clean, bordered interface for selecting a single option from multiple choices, commonly used in forms and settings panels.

### üí° Best Practices

1. **Use for single selection**: RadioList is perfect for scenarios where users need to choose one option from multiple choices.

2. **Provide clear labels**: Ensure option labels are descriptive and help users understand their choices.

3. **Use right content wisely**: Use rightContent for additional context like prices, status, or metadata.

4. **Consider width**: Set appropriate width based on your layout and content length.

5. **Handle disabled state**: Use the disabled prop when the entire list should be non-interactive.

6. **Group related options**: Keep related options together and use clear, logical groupings.
