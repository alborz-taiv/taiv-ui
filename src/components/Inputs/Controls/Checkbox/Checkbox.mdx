# Checkbox

[View Mantine Docs](https://v6.mantine.dev/core/checkbox/)

The Checkbox component is a customizable wrapper around Mantine's Checkbox component that provides consistent styling and behavior across your application. It extends Mantine v6's CheckboxProps while adding custom sizing options and preset styling.

### üì¶ Import


### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what they're agreeing to or selecting.

2. **Use controlled state**: Always manage the checked state through props for predictable behavior.

3. **Error handling**: Use the error prop to display validation messages at a field level when needed.

4. **Group related options**: When using multiple checkboxes, group them logically.
