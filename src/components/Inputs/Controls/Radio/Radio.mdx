# Radio

[View Mantine Docs](https://v6.mantine.dev/core/radio/)

The Radio component is a customizable wrapper around Mantine's Radio component that provides consistent styling and behavior across your application. It extends Mantine v6's RadioProps while adding preset styling for form inputs.

### üí° Best Practices

1. **Use in groups**: Radio buttons should always be used in groups where only one option can be selected.

2. **Provide clear labels**: Always include descriptive labels to help users understand their options.

3. **Use controlled state**: Always manage the selected value through props for predictable behavior.

4. **Error handling**: Use the error prop to display validation messages when needed.

5. **Accessibility**: Ensure labels are descriptive and consider using description for additional context.

6. **Group related options**: When using multiple radio groups, separate them clearly and consider using fieldsets for screen readers.
