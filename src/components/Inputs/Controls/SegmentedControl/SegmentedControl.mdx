# SegmentedControl

[View Mantine Docs](https://v6.mantine.dev/core/segmented-control/)

<AIGeneratedBanner />

The SegmentedControl component is a wrapper around Mantine’s SegmentedControl that aligns typography and colors with the rest of Taiv UI. It extends Mantine v6’s props while adding preset sizes that map to slightly larger Mantine sizes (same approach as Checkbox) for visual consistency without a global Mantine theme provider.

### 📦 Import

```tsx
import { SegmentedControl } from '@taiv/ui';
```

## <br />

### 💡 Best Practices

1. **Controlled usage**: Prefer `value` and `onChange` so selection stays in sync with your app state.

2. **Short labels**: Keep segment labels concise; truncation uses ellipsis when space is tight.

3. **Mutually exclusive choices**: Use SegmentedControl for a small set of exclusive options; for many options, consider Select or Tabs.

4. **Meaningful values**: Use stable `value` strings suitable for APIs and analytics, not display text.

5. **Disabled segments**: Use per-item `disabled` when an option is temporarily invalid, and `disabled` on the root when the whole control is unavailable.
