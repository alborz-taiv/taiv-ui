# Toggle

[View Mantine Docs](https://v6.mantine.dev/core/switch/)

The Toggle component is a customizable wrapper around Mantine's Switch component that provides consistent styling and behavior across your application. It extends Mantine v6's SwitchProps while adding custom sizing options and preset styling with smooth animations.

### üì¶ Import


### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what the toggle controls.

2. **Use controlled state**: Always manage the checked state through props for predictable behavior.

3. **Immediate feedback**: Toggles should take effect immediately without requiring a save action.

4. **Visual consistency**: Use consistent sizing across similar contexts in your application.

5. **Error handling**: Use the error prop to display validation messages when needed.

6. **Accessibility**: Ensure labels are descriptive and consider using description for additional context.

7. **Binary choices**: Use toggles for simple on/off states, not for multiple choice selections.
