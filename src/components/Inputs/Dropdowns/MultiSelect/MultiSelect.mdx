# MultiSelect

[View Mantine Docs](https://v6.mantine.dev/core/multi-select/)

The MultiSelect component is a customizable wrapper around Mantine's MultiSelect component that provides consistent styling and behavior across your application. It extends Mantine v6's MultiSelectProps while adding custom sizing options and preset styling with built-in search functionality.

### üì¶ Import

```tsx
import { MultiSelect } from '@taiv/ui';
```

### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what they're selecting.

2. **Handle empty states**: Use meaningful placeholders and consider the clearable prop for better UX.

3. **Search functionality**: Search is enabled by default to help users find options quickly in large datasets.

4. **Limit selections**: Use maxSelectedValues when appropriate to prevent overwhelming selections.

5. **Error handling**: Use the error prop to display validation messages when needed.

6. **Data structure**: Ensure your data array follows the `{value: string, label: string}` format for consistency.

7. **Creation feature**: Use creatable sparingly and provide clear feedback when new items are created.

8. **Performance**: For very large datasets, consider implementing virtualization or server-side filtering.
