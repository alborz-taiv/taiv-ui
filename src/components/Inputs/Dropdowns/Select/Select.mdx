# Select

[View Mantine Docs](https://v6.mantine.dev/core/select/)

The Select component is a customizable wrapper around Mantine's Select component that provides consistent styling and behavior across your application. It extends Mantine v6's SelectProps while adding custom sizing options and preset styling.

### üì¶ Import

```tsx
import { Select } from '@taiv/ui';
```

## <br />

### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what they're selecting.

2. **Handle empty states**: Use meaningful placeholders and consider the clearable prop for better UX.

3. **Search functionality**: Enable searchable for large datasets to help users find options quickly.

4. **Error handling**: Use the error prop to display validation messages when needed.

5. **Data structure**: Ensure your data array follows the `{value: string, label: string}` format for consistency.
