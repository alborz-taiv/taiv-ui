# UnstyledButton

[View Mantine Docs](https://v6.mantine.dev/core/unstyled-button/)

The UnstyledButton component is a minimal wrapper around Mantine's UnstyledButton component that provides a clean slate for custom button implementations. Unlike the styled Button component, UnstyledButton has no default styling, giving you complete control over appearance.

### üì¶ Import

### üí° Best Practices

1. **Wrap clickable elements with UnstyledButton**: Instead of adding just an onClick prop, wrap the element in an UnstyledButton to ensure proper accessibility.

2. **Provide all styles**: Since there are no defaults, ensure you style all necessary states (hover, focus, active, disabled).

3. **Use styled Button when possible**: For standard button needs, use the styled Button component for consistency.
