# TextInput

[View Mantine Docs](https://v6.mantine.dev/core/text-input/)

The TextInput component is a customizable wrapper around Mantine's TextInput component that provides consistent styling and behavior across your application. It extends Mantine v6's TextInputProps while adding custom sizing options and preset styling.

### üì¶ Import

```tsx
import { TextInput } from '@taiv/ui';
```

### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what input is expected.

2. **Use appropriate input types**: Set the correct type (email, password, tel, etc.) for better UX and validation.

3. **Add helpful placeholders**: Use placeholder text to provide examples or additional context.

4. **Handle validation**: Use the error prop to display validation messages clearly.

5. **Consider accessibility**: Ensure labels are properly associated and consider using description for additional context.

6. **Icon usage**: Use icons to provide visual context but don't rely on them alone for meaning.

7. **Width considerations**: Use custom width when needed, but be mindful of responsive design.

8. **Required fields**: Use the required prop and withAsterisk for clear indication of mandatory fields.

9. **Consistent sizing**: Use consistent sizing across related forms for better visual hierarchy.
