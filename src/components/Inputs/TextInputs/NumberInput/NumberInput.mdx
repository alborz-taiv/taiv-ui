# NumberInput

[View Mantine Docs](https://v6.mantine.dev/core/number-input/)

The NumberInput component is a customizable wrapper around Mantine's NumberInput component that provides numeric input with stepper controls and consistent styling. It extends Mantine v6's NumberInputProps while adding custom sizing options and preset styling.

### üì¶ Import

```tsx
import { NumberInput } from '@taiv/ui';
```

### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what numeric value is expected.
2. **Use helpful placeholders**: Use placeholders to give users hints about valid ranges or formats.
3. **Handle validation**: Use the error prop to display validation messages when values are invalid or required.
4. **Set appropriate ranges**: Use min and max to enforce valid ranges and prevent invalid submissions.
5. **Choose logical steps**: Set the step value based on the expected adjustment size (e.g., 1 for counts, 0.5 for ratings).
6. **Control precision**: Use precision for decimal values to avoid confusing rounding behavior.
7. **Accessibility**: Ensure labels are descriptive and consider using description for additional context.
8. **Consistent sizing**: Use consistent sizes across related forms for better visual hierarchy.
9. **Form integration**: The component works seamlessly with form libraries and validation frameworks.
10. **Responsive design**: Use the fullWidth and width props to adapt to different layouts and containers.
