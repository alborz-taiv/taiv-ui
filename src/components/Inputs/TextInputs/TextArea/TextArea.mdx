# TextArea

[View Mantine Docs](https://v6.mantine.dev/core/textarea/)

The TextArea component is a customizable wrapper around Mantine's Textarea component that provides consistent styling and behavior across your application. It extends Mantine v6's TextareaProps while adding custom sizing options and preset styling with auto-resize functionality.

### üì¶ Import


### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what content is expected.

2. **Use helpful placeholders**: Use placeholders to give users hints about the expected content format or length.

3. **Handle validation**: Use the error prop to display validation messages when content is invalid or required.

4. **Consider auto-resize**: Use autosize for content that varies significantly in length to improve user experience.

5. **Set appropriate row limits**: Use minRows and maxRows to provide consistent sizing and prevent excessive growth.

6. **Consistent sizing**: Choose appropriate sizes based on your form layout and user context.

7. **Accessibility**: Ensure labels are descriptive and consider using description for additional context.

8. **Form integration**: The component works seamlessly with form libraries and validation frameworks.

9. **Responsive design**: Use fullWidth prop when the textarea should take the full width of its container.

10. **Custom styling**: Use the styles prop to customize appearance while maintaining consistency with your design system.
