# SearchBar

[View Mantine Docs](https://v6.mantine.dev/core/text-input/)

The SearchBar component is a specialized wrapper around the custom TextInput component that provides search functionality with a built-in search icon. It extends TextInput with search-specific styling and behavior, offering a consistent search interface across your application.


### üí° Best Practices

1. **Provide clear labels**: Always include descriptive labels to help users understand what they can search for.

2. **Use helpful placeholders**: Use placeholder text to provide context about what can be searched.

3. **Handle validation**: Use the error prop to display validation messages when search queries are invalid.

4. **Consider accessibility**: Ensure labels are descriptive and consider using description for search scope.

5. **Custom width**: Use the width prop when you need specific sizing requirements for your search interface.

6. **Consistent sizing**: Choose appropriate sizes based on your layout and user context.

7. **Search icon**: The built-in search icon provides clear visual indication of search functionality.

8. **Form integration**: The component works seamlessly with form libraries and validation frameworks.

9. **Responsive design**: Use fullWidth prop when the search bar should take the full width of its container.

10. **Custom styling**: Use the styles prop to customize appearance while maintaining consistency with your design system.
