# Center

[View Mantine Docs](https://v6.mantine.dev/core/center/)

Centers its children both vertically and horizontally. This is a super useful building block along with [Stack](?path=/docs/components-layout-stack--docs) and [Group](?path=/docs/components-layout-group--docs).

### 📦 Import
```tsx
import { Center } from '@taiv/ui';
```
