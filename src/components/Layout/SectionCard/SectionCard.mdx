# SectionCard

An extension of [Card](?path=/docs/components-layout-card--docs) that adds a built-in title and subtitle at the top.

### 📦 Import

```tsx
import { SectionCard } from '@taiv/ui';
```

### ✍️ Basic Usage

<Canvas of={SectionCardStories.Default} />
<Controls of={SectionCardStories.Default} />

---

### Animation

Enable a lifting hover animation with the `animate` prop. This is especially useful when the card represents an "item" of some kind.

<Story of={SectionCardStories.Animation} />
