# Stack

[View Mantine Docs](https://v6.mantine.dev/core/stack/)

A vertical flex container that arranges children in a column. This is an essential building block for creating layouts, along with [Group](?path=/docs/components-layout-group--docs) and [Center](?path=/docs/components-layout-center--docs).

### 📦 Import
```tsx
import { Stack } from '@taiv/ui';
```

### Positioning Within Stacks

`justify` controls vertical alignment. `align` controls horizontal alignment, but using a [Group](?path=/docs/components-layout-group--docs) for that is generally preferred.
