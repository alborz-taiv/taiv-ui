# Tabs

[View Mantine Docs](https://v6.mantine.dev/core/tabs/)

Organizes content into sections that users can switch between.

### 📦 Import
```tsx
import { Tabs } from '@taiv/ui';
```

### ✍️ Basic Usage

<Canvas of={TabsStories.Default} />
<Controls of={TabsStories.Default} />

The Tabs component is 'composed' of `Tabs.List`, `Tabs.Tab`, and `Tabs.Panel`. These are linked via a matching `value` prop.

```tsx
<Tabs defaultValue="overview">
  // First, we list the tabs that are available to use
  <Tabs.List>
    <Tabs.Tab value="overview">Overview</Tabs.Tab>
    <Tabs.Tab value="details">Details</Tabs.Tab>
    <Tabs.Tab value="settings">Settings</Tabs.Tab>
  </Tabs.List>

  // Then, we define the content for each tab
  <Tabs.Panel value="overview">...</Tabs.Panel>
  <Tabs.Panel value="details">...</Tabs.Panel>
  <Tabs.Panel value="settings">...</Tabs.Panel>
</Tabs>
```

## <br />

### 💡 Tips & Best Practices

1. **Use semantic tab values.** Name values by content (`"settings"`, `"profile"`), not position (`"tab1"`, `"tab2"`).
2. **We support keyboard navigation.** You can use the left and right arrows to navigate between tabs.
