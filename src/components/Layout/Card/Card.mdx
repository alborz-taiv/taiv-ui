# Card

[View Mantine Docs](https://v6.mantine.dev/core/card/)

A flexible container for grouping related content into one space. You can use this as-is or extend it for other components.

We also have other cards available - for example, see [SectionCard](?path=/docs/components-layout-sectioncard--docs).

### 📦 Import
```tsx
import { Card } from '@taiv/ui';
```
