# Group

[View Mantine Docs](https://v6.mantine.dev/core/group/)

A horizontal flex container that arranges children in a row. This is an essential building block for creating layouts, along with [Stack](?path=/docs/components-layout-stack--docs) and [Center](?path=/docs/components-layout-center--docs).

### 📦 Import
```tsx
import { Group } from '@taiv/ui';
```

### Positioning Within Groups

`position` controls horizontal alignment. `align` controls vertical alignment, but using a [Stack](?path=/docs/components-layout-stack--docs) for that is generally preferred.

<Story of={GroupStories.PositionAlign} />

## <br />

### Grow Items

`grow` makes children expand to fill available space. 

This is useful for many use cases, like button groups or mobile layouts where filling the screen is important.

<Story of={GroupStories.GrowItems} />

## <br />

### Wrap Items

`noWrap` prevents items from wrapping to new lines. This is useful if things need to be restricted to a single line.
