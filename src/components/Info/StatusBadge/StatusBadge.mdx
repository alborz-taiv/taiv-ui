# StatusBadge

A pill-shaped badge for showing **online** or **offline** status (for now, more can be added). You can override the title, icon, background color, and foreground (title) color.

## Usage

```tsx
import { StatusBadge } from '@taiv/ui';

// Default variants
<StatusBadge variant="online" />
<StatusBadge variant="offline" />

// With custom title
<StatusBadge variant="online" title="Available" />
<StatusBadge variant="offline" title="Away" />

// With custom colors
<StatusBadge variant="online" backgroundColor="#e8f5e9" color="#2e7d32" />

// With custom icon
<StatusBadge variant="offline" icon={<MyIcon size={20} />} />
```

## Variants

- **Online** — Green styling for “available” or “connected”
- **Offline** — Red styling for “unavailable” or “disconnected”

<Group gap="1.6rem" style={{ marginTop: '1rem' }}>
  <StatusBadge variant="online" />
  <StatusBadge variant="offline" />
</Group>

## Default size

- Width: **9rem**
- Height: **3rem**
- Border radius: **10rem** (pill shape)

## Custom size

Use the `width` and `height` props to adjust the badge size:

<Group gap="1.6rem" align="flex-end" style={{ marginTop: '1rem' }}>
  <StatusBadge variant="online" width="6rem" height="1.8rem" />
  <StatusBadge variant="offline" />
  <StatusBadge variant="online" width="9rem" height="2.8rem" />
</Group>

## Override title

Pass the `title` prop to show a custom label instead of the default "Online" / "Offline":

<Group gap="1.6rem" style={{ marginTop: '1rem' }}>
  <StatusBadge variant="online" title="Available" />
  <StatusBadge variant="offline" title="Away" />
  <StatusBadge variant="online" title="Connected" />
</Group>

## Override icon

Pass the `icon` prop to use a custom icon instead of the default variant icon:

<Group gap="1.6rem" style={{ marginTop: '1rem' }}>
  <StatusBadge variant="online" icon={<IconMoodHappy size={20} />} />
  <StatusBadge variant="offline" icon={<IconMoodHappy size={20} />} />
</Group>

## Override colors

Use `backgroundColor` and `color` to override the variant’s background and foreground (title and icon) colors:

<Group gap="1.6rem" style={{ marginTop: '1rem' }}>
  <StatusBadge variant="online" backgroundColor="black" color="white" />
  <StatusBadge variant="offline" backgroundColor="black" color="white" />
</Group>
