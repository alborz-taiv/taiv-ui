# Badge

[View Mantine Docs](https://v6.mantine.dev/core/badge/)

The Badge component is a customizable wrapper around Mantine's Badge component that provides consistent styling and behavior across your application. It extends Mantine v6's BadgeProps while adding custom variants, sizing options, and design system colors.

Use Badge for labels, tags, and status indicators. See variants and colors below for different use cases.

### üì¶ Import


## <br />

### üí° Best Practices

1. **Use semantic colors**: Choose colors that match the meaning (e.g. success for completed, error for failed).

2. **Don't type out defaults**: The default variant is outline, default size is md, and default color is primary. Omit these props when they're sufficient.

3. **Use leftIcon for status**: For status indicators (e.g. online/offline), use the `leftIcon` prop with a filled circle or similar icon.

4. **Keep labels short**: Badges work best with one or two words.

5. **Match size to context**: Use sm in tables or dense UIs, md for most cases, lg when the badge needs to stand out.
