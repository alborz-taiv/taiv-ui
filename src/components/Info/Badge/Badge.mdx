# Badge

A customizable badge built on [Mantine Badge](https://v6.mantine.dev/core/badge/). Use it for labels, tags, and status indicators. Supports preset sizes, colors from the design system, outline/filled/gradient variants, and an optional **left icon**.

## Usage

```tsx
import { Badge } from '@taiv/ui';

// Variants: outline (default), filled, gradient
<Badge variant="outline" color="primary">Label</Badge>
<Badge variant="filled" color="success">Success</Badge>
<Badge variant="gradient" color="primary">Gradient</Badge>

// Sizes: sm, md (default), lg
<Badge size="sm">Small</Badge>
<Badge size="md">Medium</Badge>
<Badge size="lg">Large</Badge>

// Colors: primary, success, warning, error, neutral, white
<Badge color="primary">Primary</Badge>
<Badge color="error">Error</Badge>

// Optional left icon (e.g. filled circle)
<Badge leftIcon={<IconCircleFilled size={14} />} color="success">Online</Badge>
```

## Variants

- **Outline** — Border and text in the chosen color, transparent background (default)
- **Filled** — Background and text in the chosen color
- **Gradient** — Gradient from light to main color, white text
