# FormModal

[View Mantine Docs](https://v6.mantine.dev/core/modal/)


The FormModal component is a structured modal built on Mantine's Modal, with an optional icon, content area, and configurable cancel/confirm footer buttons. Use it for confirmations, forms, or short messages that require user action.

### üí° Best Practices

1. **Control visibility with state**: Use `useState` for `opened` and wire `onClose`, `handleCancel`, and `handleConfirm` to close the modal when appropriate.

2. **Use semantic variants**: Choose `modalVariant` to match intent (e.g. `cancel` for destructive actions, `success` for positive confirmations).

3. **Override labels when needed**: Use `cancelLabel` and `confirmLabel` for context-specific copy; otherwise rely on variant defaults.

4. **Keep content focused**: Use `children` for concise messages or simple forms. Consider dedicated layout components for complex forms.

5. **Width and responsive behavior**: Use `width` for consistency with your layout. Test at different viewport sizes when using fixed widths.
