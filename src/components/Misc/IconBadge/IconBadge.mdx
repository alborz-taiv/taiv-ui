# IconBadge

The IconBadge component is just a colored icon wrapped in a circular background. It can be sized dynamically to fit any use case.

### 📦 Import

```tsx
import { IconBadge } from '@taiv/ui';
```

### ✍️ Basic Usage

<Canvas of={IconBadgeStories.Default} />
<Controls of={IconBadgeStories.Default} />

In the icon that you pass in, use `size` to dynamically control the size of the badge as a whole:

```tsx
<IconBadge icon={<IconHeart size={20} />}/>
```
