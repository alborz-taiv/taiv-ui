# Fraction

The Fraction component renders a numerator and denominator separated by a horizontal line. It extends <a href="/?path=/docs/components-typography-text--docs">Text</a> styling, so it supports the same variants and overrides.

### 📦 Import

```tsx
import { Fraction } from '@taiv/ui';
```
