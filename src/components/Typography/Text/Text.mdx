# Text

[View Mantine Docs](https://v6.mantine.dev/core/text/)

The Text component utilizes the variants we have from our <a href="/?path=/docs/design-fonts--docs">font constants</a>, and allows us to easily render text onscreen using our design system.

We use anywhere we need body text rendered, whether that's in a UI component or just on a page in general. For headers, see <a href="/?path=/docs/components-typography-title--docs">Title</a> instead.

### 📦 Import

```tsx
import { Text } from '@taiv/ui';
```
