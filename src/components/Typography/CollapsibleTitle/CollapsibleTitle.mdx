# CollapsibleTitle

CollapsibleTitle shows a title and subtext with a caret; clicking toggles the expanded state and reveals optional children. Use it for section headers that expand to show more content (e.g. settings sections, accordions). The title uses medium-weight header styling and the subtext uses caption styling by default; colors follow your typography tokens.

### 📦 Import

```tsx
import { CollapsibleTitle } from '@taiv/ui';
```

### ✍️ Basic Usage

Control open state with `opened` and `setOpened`. Pass content as `children` to show when expanded.

### 💡 Best practices

1. **Control state at the parent**: Keep `opened` and `setOpened` in the parent so you can sync with URL, form state, or other UI.

2. **Use for sections, not one-off toggles**: Prefer CollapsibleTitle for "section header + expandable block". For a single line of tappable text, use CollapsibleText.

3. **Keep titles and subtext short**: The header is two lines (title + subtext); avoid long copy so the caret stays aligned and the block stays scannable.
