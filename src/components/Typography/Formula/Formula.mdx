# Formula

The Formula component automatically parses a mathematical expression (string) and renders it onscreen using <a href="/?path=/docs/components-typography-text--docs">Text</a> and <a href="/?path=/docs/components-typography-fraction--docs">Fraction</a>.

### 📦 Import

```tsx
import { Formula } from '@taiv/ui';
```

### Operations

Pass a plain string expression — the component parses it and formats each part accordingly. There are a number of different operations we can parse.

<Canvas of={FormulaStories.Operations} />

<table>
  <thead>
    <tr>
      <th>Syntax</th>
      <th>Operation</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>a + b</code></td>
      <td>Addition</td>
    </tr>
    <tr>
      <td><code>a - b</code></td>
      <td>Subtraction</td>
    </tr>
    <tr>
      <td><code>a * b</code></td>
      <td>Multiplication</td>
    </tr>
    <tr>
      <td><code>a / b</code></td>
      <td>Division</td>
    </tr>
    <tr>
      <td><code>a_b</code></td>
      <td>Subscript</td>
    </tr>
  </tbody>
</table>

Currently, superscripts (exponents) and square roots are not supported.
