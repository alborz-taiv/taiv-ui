# CollapsibleText

<AIGeneratedBanner />

CollapsibleText shows a single line of text with a caret; clicking toggles the expanded state and reveals optional children. Use it for simple expand/collapse toggles (e.g. "Show more", inline details). The label uses body text styling by default; use the `variant` prop to match your hierarchy. For a two-line header (title + subtext), use CollapsibleTitle instead.

### 📦 Import

```tsx
import { CollapsibleText } from '@taiv/ui';
```

### ✍️ Basic Usage

Control open state with `opened` and `setOpened`. Pass content as `children` to show when expanded.

## <br />

### 💡 Best practices

1. **Control state at the parent**: Keep `opened` and `setOpened` in the parent so you can sync with URL, form state, or other UI.

2. **Use for single-line toggles**: Prefer CollapsibleText when you only need one line of clickable text. For a section header with title + subtext, use CollapsibleTitle.

3. **Keep the label short**: One line of text keeps the caret aligned and the control easy to scan.
