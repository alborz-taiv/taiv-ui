# Title

[View Mantine Docs](https://v6.mantine.dev/core/title/)

The Title component utilizes the variants we have from our <a href="/?path=/docs/design-fonts--docs">font constants</a>, and renders semantic HTML heading elements using our design system.

We use it anywhere we need a heading or subheading — page titles, section headers, card headers, etc. For body text, see <a href="/?path=/docs/components-typography-text--docs">Text</a> instead.

### 📦 Import

```tsx
import { Title } from '@taiv/ui';
```
