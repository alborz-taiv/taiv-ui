# Chart

[Recharts documentation](https://recharts.org/)

The Chart component composes Recharts line, area, and bar series on a single `ComposedChart` with shared axes, grid, tooltip, and optional legend. Axis tick labels and tooltip values are formatted with `getChartFormatter` using `yAxisFormat` / `xAxisFormat` (`format` plus optional `decimalPlaces` and `truncateAt`).

Use Chart for time-series or categorical metrics where multiple series share the same X categories. For pie or donut data, use **PieChart**.

### 📦 Import

```tsx
import { Chart } from '@taiv/ui';
```

## <br />

### 💡 Best practices

1. **Align series keys** — Every series must use the same ordered `key` values so Recharts rows line up.

2. **Pick X format from data** — Use `string` for real categories; use numeric formats only when keys represent amounts or numeric buckets.

3. **Match Y format to semantics** — Use `currency` for revenue, `percentage` for rates, `integer` for counts.

4. **Truncate large magnitudes** — Use `truncateAt` for dashboards where axis space is tight.

5. **Give the chart a height** — When using `height="100%"`, ensure an ancestor sets a definite height (Storybook examples use a fixed `Box` height).
