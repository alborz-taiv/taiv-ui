# Chart

[Recharts documentation](https://recharts.org/)

The Chart component composes Recharts line, area, and bar series on a single `ComposedChart` with shared axes, grid, tooltip, and optional legend. Tick labels and tooltip values go through `getChartFormatter`.

**`yAxisFormat` and `xAxisFormat`** each accept either:

- A **format key string** — any key of `formats` (`string`, `integer`, `decimal`, `currency`, `percentage`, `multiple`). Defaults match this shape: **`yAxisFormat`** defaults to **`'decimal'`**, **`xAxisFormat`** to **`'string'`**.
- An **object** — `{ format, options? }` where `options` may include `decimalPlaces` and `truncateAt` (`thousand` | `million`), per `ChartFormatOptions`.

Internally, the chart runs both through **`parseFormat`**: strings become `{ format, options: {} }`; objects keep `format` and use `options ?? {}`.

Use Chart for time-series or categorical metrics where multiple series share the same X categories. For pie or donut data, use **PieChart**.

### 📦 Import

```tsx
import { Chart } from '@taiv/ui';
```

### ✍️ Basic usage

The **Default** story uses string shorthands (`yAxisFormat="decimal"`, `xAxisFormat="string"`). Controls offer a select per axis for format keys; for **`decimalPlaces`** or **`truncateAt`**, use the object form in code (see **YAxisDecimalPlaces**, **YAxisTruncation**, **XAxisNumericFormats**).

```tsx
<Chart series={series} yAxisFormat="currency" xAxisFormat="string" />

<Chart
  series={series}
  yAxisFormat={{ format: 'decimal', options: { decimalPlaces: 2, truncateAt: 'million' } }}
/>
```

### 🎛️ Y-axis options (`options` on object form)

When `yAxisFormat` is an object, **`options`** can include:

- **decimalPlaces** — Overrides the default decimal count for that format (see **YAxisDecimalPlaces**).
- **truncateAt** — **thousand** or **million**: values at or above the threshold render with a compact suffix (`k` / `M`). See **YAxisTruncation**.

## <br />

### 💡 Best practices

1. **Align series keys** — Every series must use the same ordered `key` values so Recharts rows line up.

2. **Prefer string shorthands** — Use `yAxisFormat="currency"` when defaults are enough; use the object form only for `decimalPlaces` or `truncateAt`.

3. **Pick X format from data** — Use `string` for real categories; use numeric formats only when keys represent amounts or numeric buckets.

4. **Match Y format to semantics** — Use `currency` for revenue, `percentage` for rates, `integer` for counts.

5. **Truncate large magnitudes** — Pass `{ format, options: { truncateAt: 'million' } }` (or `thousand`) when axis space is tight.

6. **Give the chart a height** — When using `height="100%"`, ensure an ancestor sets a definite height (Storybook examples use a fixed `Box` height).
