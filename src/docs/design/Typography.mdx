# Fonts

All typography in Taiv UI uses **Poppins** as the base font family. 

There are a few different typography components, but the main ones are <a href="/?path=/docs/components-typography-title--docs">Title</a> and <a href="/?path=/docs/components-typography-text--docs">Text</a>. 

Both of these contain a set of variants for common use cases. Every typography component can also have its size, weight, and color overridden to fit more specific mockups that dont cleanly fit a variant. 

<br />

<br />

### ✏️ Titles

Titles are used for headings and section labels. Each variant maps to a semantic HTML heading element.

### ✏️ Text

Text is used for body copy, component labels, content, etc. We have some named variants that are all `<p>` tags under the hood, for common use cases.

<Stack gap="20px" style={{ border: `1px dashed ${neutral[100]}`, borderRadius: '8px', padding: '20px' }}>
  <Stack gap="2.5px">
    <Text variant="body">Body — The quick brown fox jumps over the lazy dog.</Text>
    <Text variant="label">variant="body" — 14px / medium / neutral[200]</Text>
  </Stack>
  <Stack gap="2.5px">
    <Text variant="subtle">Subtle — The quick brown fox jumps over the lazy dog.</Text>
    <Text variant="label">variant="subtle" — 14px / regular / neutral[200]</Text>
  </Stack>
  <Stack gap="2.5px">
    <Text variant="label">Label — The quick brown fox jumps over the lazy dog.</Text>
    <Text variant="label">variant="label" — 12px / medium / neutral[200]</Text>
  </Stack>
  <Stack gap="2.5px">
    <Text variant="caption">Caption — The quick brown fox jumps over the lazy dog.</Text>
    <Text variant="label">variant="caption" — 10px / regular / neutral[200]</Text>
  </Stack>
</Stack>
