# Spacing

We have a consistent set of spacing tokens used across all components for padding, margin, gap, etc.

For now, use these only when developing components *within* Taiv UI. For consuming projects, see [Frontend Standards](?path=/docs/getting-started-frontend-standards--docs).

<br />

### 📦 Import

```tsx
// Import either spacing as a whole or an individual token
import { spacing, lg} from '@taiv/ui/constants';

// Use as an object
gap: spacing.lg  

// Alternatively, use an individual token
padding: lg
```
