# **Helpful Links**

Here are some helpful links to make the most out of using Taiv UI:

- [Figma](https://www.figma.com/design/vyALUIBVWvYzDlKfxKp1Ye/Master?node-id=272-8344&t=1QyQQIOk4WMX8tKl-0) - Design mocks for this toolkit
- [Mantine v6](https://v6.mantine.dev/) - For complex use cases and additional styling/props that this documentation does not cover (but supports)
- [Mantine Prebuilts](https://ui.mantine.dev/) - For inspiration on new components and extended styling/props
- [FontAwesome Icons](http://fontawesome.com/v6/search?ic=free&o=r) - One set of icon options
- [Tabler Icons](https://tabler.io/icons) - Another set of icon options
- [GitHub](https://github.com/aydenoffenloch/taiv-ui-toolkit) - The repository for this toolkit
