# **Helpful Links**

Here are some helpful links to make the most out of using Taiv UI:

- [Figma](https://www.figma.com/design/yx4KagX6prHx7T7LerCNOd/Taiv-x-Mantine) - Design mocks for this toolkit
- [Mantine v6](https://v6.mantine.dev/) - For complex use cases and additional styling/props that this documentation does not cover (but supports)
- [Mantine Prebuilts](https://ui.mantine.dev/) - For inspiration on new components and extended styling/props
- [Tabler Icons](https://tabler.io/icons) - Our primary set of icons
- [FontAwesome Icons](http://fontawesome.com/v6/search?ic=free&o=r) - Our secondary set of icons
- [Lunch & Learn Presentation](https://docs.google.com/presentation/d/1fq9vcvNN8drmYhFh4jqXx-7Cw-sCCNN_gocPyc0tJeY) - The Lunch & Learn presentation for this toolkit (from Jan. 2026)
- [GitHub](https://github.com/alborz-taiv/taiv-ui) - The repository for this toolkit
